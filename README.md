# Vue 3 + Vite

Esse é um projeto simples de lista de tarefas, desenvolvi para treinar conceitos e lõgica do Vue.js, como reatividade, data binding e eventos. O projeto salva as tarefas no LocalStorage.

O projeto foi criado utilizando apenas HTML + CSS + JavaScript + Vue.js via CDN, sem build tools

***A aplicação é estruturada assim:***

---> Um campo de texto para digitar novas tarefas

---> Uma lista que exibe todas as tarefas adicionadas

---> Botões para remover e marcar como concluída

---> Salvamento automático no localStorage sempre que a lista é alterada

***Funcionalidades***

---> Adicionar tarefa

---> Excluir tarefa

---> Marcar como concluída

---> Estilização personalizada (tema lilás, fonte Courier)

---> Salvamento automático no localStorage

---> Interface simples, responsiva e fácil de entender


This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.
